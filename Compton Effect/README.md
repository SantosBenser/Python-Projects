# Compton Shift as a Function of Scattering Angle

## 📌 Title

Compton Shift as a Function of Scattering Angle Using Python

A Python-based scientific visualization project that demonstrates the relationship between scattering angle and Compton wavelength shift. The project implements the Compton Scattering equation and visualizes how the wavelength of scattered photons changes as the scattering angle varies from 0° to 180°.

## ⚙️ The Process

### 1. Define Physical Constants

The program initializes:

* Planck's Constant (h)
* Electron Rest Mass (me)
* Speed of Light (c)

### 2. Calculate Compton Wavelength

The Compton wavelength is calculated using:

λc = h / (me × c)

### 3. Generate Scattering Angles

A range of scattering angles from:

0° to 180°

is generated for analysis.

### 4. Compute Compton Shift

The Compton scattering equation is applied to determine the wavelength shift corresponding to each scattering angle.

### 5. Unit Conversion

The calculated wavelength shifts are converted from meters to picometers (pm) for easier visualization.

### 6. Plot Results

The relationship between scattering angle and Compton shift is plotted using Matplotlib.

## 📊 Theory

### Compton Scattering Equation

The wavelength shift is given by:

Δλ = (h / me c)(1 − cos θ)

Where:

* Δλ = Change in wavelength
* h = Planck's Constant
* me = Electron Rest Mass
* c = Speed of Light
* θ = Scattering Angle

### Significance

Compton Scattering provided strong evidence that light behaves as particles (photons) and played a major role in the development of quantum mechanics.
